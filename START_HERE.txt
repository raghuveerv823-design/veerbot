📋 3 SIMPLE STEPS (बस यही करो):

STEP 1️⃣: GitHub पर अपना account + repository बना
──────────────────────────────────────────────────────────────────────────────────────

a) जाओ: https://github.com/signup
b) Sign up कर (email से)
c) नया repository बना:
   - जाओ: https://github.com/new
   - Name: veerbot
   - Public रख
   - Create करो


STEP 2️⃣: अपना code GitHub पर upload करो
──────────────────────────────────────────────────────────────────────────────────────

Terminal खोल (Command Prompt या PowerShell) और चला:

cd c:\Users\user\Downloads\veerbot

git config --global user.name "Your Name"
git config --global user.email "your-email@gmail.com"

git init
git add .
git commit -m "VEERBOT Cloud Deployment"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/veerbot.git
git push -u origin main

⚠️ IMPORTANT: YOUR-USERNAME को अपना GitHub username से बदलो

Example: अगर तुम्हारा GitHub username "raj_12" है तो:
git remote add origin https://github.com/raj_12/veerbot.git


STEP 3️⃣: Render.com पर Deploy करो (1-2 मिनट में होगा)
──────────────────────────────────────────────────────────────────────────────────────

a) जाओ: https://render.com
b) "Sign up" करो (GitHub से)
c) "New +" बटन दबाओ
d) "Web Service" चुनो
e) अपना veerbot repository connect करो
f) Fill करो:
   - Name: veerbot (या कोई नाम)
   - Runtime: Python 3.11
   - Build Command: pip install -r requirements.txt
   - Start Command: gunicorn app:app
g) "Create Web Service" दबाओ
h) 3-5 मिनट wait करो
i) Deploy हो गया! तुम्हें एक URL दिया जाएगा

✅ तुम्हारा URL कुछ ऐसा होगा: https://veerbot-xyz123.onrender.com

❌ PROBLEMS SOLVING:
═══════════════════════════════════════════════════════════════════════════════════════

Problem: git command नहीं मिल रहा
Fix: https://git-scm.com/download/win से install करो

Problem: GitHub username/password गलत है
Fix: GitHub personal token use करो:
     https://github.com/settings/tokens
     Password की जगह token लगाओ

Problem: Deploy fail हो रहा है
Fix: Render dashboard पर "Logs" check करो - error दिख जाएगा

Problem: Bot का URL काम नहीं कर रहा
Fix: Render dashboard पर "health" check करो
     या 5 मिनट wait करो (deployment चल रहा हो सकता है)

Problem: Bot slow है या response नहीं दे रहा
Fix: यह normal है Render free tier पर पहली request में
     दूसरी request से fast होगा
