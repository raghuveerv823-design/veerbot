📋 स्टेप 1: Git और GitHub Setup
═══════════════════════════════════════════════════════════════════════════════════════

अगर पहली बार कर रहे हो तो:

1️⃣ Git install करो:
   Download करो: https://git-scm.com/download/win
   Install करके खोल दो

2️⃣ GitHub account बना (अगर नहीं है):
   जाओ: https://github.com/signup
   Sign up करो कोई email से

3️⃣ Terminal खोल (Command Prompt या PowerShell)

4️⃣ तीनों command अलग-अलग चला:

   git config --global user.name "Your Name"
   git config --global user.email "your-email@gmail.com"
   git config --global user.password "your-github-password"

═══════════════════════════════════════════════════════════════════════════════════════

📋 स्टेप 4: Render.com पर Deploy करो (फ्री Cloud)
═══════════════════════════════════════════════════════════════════════════════════════

Render सबसे आसान है, फ्री है, और 24/7 चलता है!

1️⃣ जाओ: https://render.com

2️⃣ Sign up करो (GitHub से sign up करो - आसान होगा)

3️⃣ Dashboard पर "New +" क्लिक करो

4️⃣ "Web Service" चुन

5️⃣ "Connect a repository" में अपना veerbot repository select करो

6️⃣ Fill करो:
   Name: veerbot
   Runtime: Python 3.11
   Build Command: pip install -r requirements.txt
   Start Command: gunicorn app:app

7️⃣ Create Web Service क्लिक करो

8️⃣ Wait करो... 3-5 मिनट लगेंगे deploy होने में

9️⃣ Deploy हो गया! तुम्हें एक URL दिया जाएगा जैसे:
   https://veerbot-xxxxx.onrender.com
