STEP 2: Upload Code (10 min)
├─ Open: Command Prompt
└─ Run these commands:
   cd c:\Users\user\Downloads\veerbot
   git config --global user.name "Your Name"
   git config --global user.email "your-email@gmail.com"
   git init
   git add .
   git commit -m "Deploy"
   git remote add origin https://github.com/USERNAME/veerbot.git
   git push -u origin main

STEP 3: Deploy (auto, 5 min)
├─ Render dashboard → New Web Service
├─ Select veerbot repo
├─ Fill:
│  ├─ Name: veerbot
│  ├─ Runtime: Python 3.11
│  ├─ Build: pip install -r requirements.txt
│  └─ Start: gunicorn app:app
└─ Create & wait

STEP 4: Access (instant)
├─ Open: https://veerbot-xyz.onrender.com
├─ Chat!
└─ Share URL
