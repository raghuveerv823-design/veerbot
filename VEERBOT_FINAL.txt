🚀 3 EASY STEPS:
═══════════════════════════════════════════════════════════════════════════════════════


STEP 1️⃣: GitHub Setup (2 minutes)
───────────────────────────────────────────────────────────────────────────────────────

a) Sign up: https://github.com/signup

b) Create repository:
   - Go to: https://github.com/new
   - Name: veerbot
   - Public: ✓
   - Create: ✓


STEP 2️⃣: Upload Code to GitHub (Copy paste Terminal commands)
───────────────────────────────────────────────────────────────────────────────────────

Open Command Prompt/PowerShell and paste commands:

   cd c:\Users\user\Downloads\veerbot
   
   git config --global user.name "Your Name"
   git config --global user.email "your-email@gmail.com"
   
   git init
   git add .
   git commit -m "VEERBOT Cloud Deployment"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/veerbot.git
   git push -u origin main

⚠️ Replace "YOUR-USERNAME" with your actual GitHub username


STEP 3️⃣: Deploy on Render (5 minutes, auto everything)
───────────────────────────────────────────────────────────────────────────────────────

a) Go to: https://render.com
b) Sign up (choose GitHub)
c) Click "New +" → "Web Service"
d) Select veerbot repository
e) Fill:
   Name: veerbot
   Runtime: Python 3.11
   Build: pip install -r requirements.txt
   Start: gunicorn app:app
f) Click "Create Web Service"
g) Wait 3-5 minutes...
h) Done! ✅
