📍 REGISTRATION 1: GitHub
──────────────────────────────────────────────────────────────────────────────────────

What: Code repository hosting (where your code lives on cloud)
Cost: FREE!
Time: 2 minutes

Steps:
1. Go to: https://github.com/signup
2. Enter:
   - Username: कोई भी नाम (future में GitHub link में होगा)
   - Email: अपना email (verification के लिए)
   - Password: कोई भी strong password
3. Verify email (check inbox)
4. Done! ✓


Why needed:
✓ Upload code to cloud
✓ Render से connect करने के लिए
✓ Updates push करने के लिए


Example username: veerbot-official, raj_chatbot, college_bot_2024 (कोई भी)
URL बनेगा: https://github.com/YOUR-USERNAME/veerbot


📍 REGISTRATION 2: Render
──────────────────────────────────────────────────────────────────────────────────────

What: Cloud hosting (जहां bot 24/7 चलेगा)
Cost: FREE! (Free tier available)
Time: 2 minutes

Steps:
1. Go to: https://render.com
2. Click "Sign Up"
3. Choose "GitHub" (easiest)
4. Authorize (allow Render to access GitHub)
5. Done! ✓


Why needed:
✓ Bot को cloud पर host करने के लिए
✓ 24/7 online रखने के लिए
✓ Free tier सब कुछ दे देता है


What you get FREE:
✓ Web service (bot hosting)
✓ HTTPS/SSL certificate (secure)
✓ 1000 hours/month (enough for 24/7)
✓ 500MB storage
✓ Auto-redeploy from GitHub
